## Overview

This repository contains the research paper titled **"AI Recipe Generator"**.

The paper proposes an AI-based system that helps users generate recipes based on the ingredients they have. The system uses **Natural Language Processing (NLP)** and **Computer Vision** to understand user inputs and recommend personalized recipes.

## Technologies Used

* Artificial Intelligence
* Natural Language Processing (NLP)
* Computer Vision
* YOLOv5
